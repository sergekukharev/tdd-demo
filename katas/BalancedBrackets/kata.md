# Balanced Brackets kata

Write a function to determine if an expression is correctly bracketed.

Example input
```php
$input = 'a + (b * c)'; // TRUE
$input = 'a + (b * c))'; // FALSE
$input = 'a + ((b * c)'; // FALSE
$input = '{a} + (b * c)'; // TRUE
$input = '()[]{}'; // TRUE
$input = '({[})'; // FALSE
$input = '{})))'; // FALSE
```
