# TDD Demo

A repo I'm using to demonstrate TDD technique.

My goal is to teach following TDD concepts:

* 3 rules of TDD
* Red-Green-Refactor 
* The Single Assert Rule
* Arrange, Act, Assert Rule
* Principle: "As tests get more specific, code gets more generic"
